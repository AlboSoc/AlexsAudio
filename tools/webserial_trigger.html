AlexsAudio Trigger Console

This page sends fixed-size trigger packets over WebSerial to an AlexsAudio device over its USB serial port. Packet format: A5 01 cmd soundId seq checksum, where checksum is the XOR of the first five bytes.

Notes

  • Use a Chromium-based browser with WebSerial support.
  • Close any serial monitor before connecting here. Only one app can own the port at a time.
  • You can connect either directly to the sound server or to the trigger client.
  • The firmware still accepts normal text commands like play 3 on the same port.
